How to install?
==============================

Clean flash:
* Have latest OxygenOS 11 on both slots (A/B).
* Boot temporary TWRP-recovery from [@sevralt_updates](https://t.me/sevralt_updates).
* Format data and reboot recovery again.
* Put latest build to internal memory by MTP.
* Flash it.
* Reboot system!

Dirty flash:
* Boot temporary TWRP-recovery from [@sevralt_updates](https://t.me/sevralt_updates).
* Put latest build to internal memory by MTP.
* Flash it.
* Reboot system!
